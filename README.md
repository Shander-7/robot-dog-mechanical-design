# robot-dog-mechanical-design

## Overview
This project presents the preliminary mechanical design of a simple quadruped robot.

## Project Features
- plastic body frame
- Four-legged robot
- 8 Degrees of Freedom (DOF)
- Servo motors for all joints
- Initial torque calculation
- Center of gravity analysis
- Stable walking method
- Mechanical challenges and solutions
